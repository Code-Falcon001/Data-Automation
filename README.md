# Data Automation & ETL Pipelines ⚙️🕷️

This repository houses my data engineering, VBA automation, and automated pipeline developments.

## 📂 Sub-Projects

### 1. [Web-Scraping](./Web-Scraping)
* An automated Python ETL pipeline that scrapes web job postings, parses candidate CVs, ranks compatibility scores using local LLMs, and stores records in an Excel spreadsheet database.

### 2. [excel-change-tracker](./excel-change-tracker)
* An HTML/VBA utility to track edits, changes, and updates in Excel workbooks.

### 3. [excel-change-tracker-v2](./excel-change-tracker-v2)
* Version 2 of the Excel change tracking engine.

### 4. [excel-consolidation-engine](./excel-consolidation-engine)
* A config-driven Excel VBA consolidation engine designed to merge multiple worksheets and reports automatically.
